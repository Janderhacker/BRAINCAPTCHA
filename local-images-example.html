This guide shows you how to use local images in your BrainCaptcha questions.

Step 1: Create Your Questions with Local Images

1. Open the Question Designer: question-designer/index.html

2. Select "Image Question" as the question type

3. Click "üìÅ Upload Local Image" to select an image from your computer

4. The tool will suggest a path like /images/your-image.jpg

6. Click "Add Question" to save it

Step 2: Export Your Configuration

1. Click "Export with Settings" to download your complete configuration

2. This will give you a braincaptcha-config.js file

3. The file includes instructions for which images to copy where
